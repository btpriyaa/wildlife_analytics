# wildlife_analytics
A production-quality dbt project on Snowflake modelling wildlife park visitor, revenue, and encounter analytics. Built as a portfolio project to demonstrate hands-on capability with a Snowflake + dbt data stack in a wildlife park domain.
> **Data is fully synthetic.** No proprietary data is used.

---

## Stack

| Layer | Tool |
|-------|------|
| Cloud data warehouse | Snowflake (Free Trial) |
| Transformation | dbt Core 1.8 |
| Orchestration | GitHub Actions (CI/CD) |
| Version control | Git + GitHub |
| Language | SQL, Python |
